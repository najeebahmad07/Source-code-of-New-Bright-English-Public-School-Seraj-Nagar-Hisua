## Introduction

New Bright English Public School's website is designed to provide students, parents, and educators with all the necessary information about the school. The website offers a user-friendly interface with detailed sections about the school's history, curriculum, faculty, and more.

## Features

- **Responsive Design**: The website is fully responsive and works on all devices, including desktops, tablets, and mobile phones.
- **Interactive Elements**: Includes interactive elements like image sliders, dropdown menus, and contact forms.
- **Informative Sections**: Detailed sections for About Us, Admissions, Academics, Faculty, Gallery, and Contact Us.
- **News and Updates**: Section for the latest news and updates related to the school.
- **Online Admission Form**: Easy-to-use online admission form for new student enrollments.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling the website.
- **JavaScript**: For interactive features and dynamic content.
- **Bootstrap**: For responsive design and layout.
- **PHP**: For server-side scripting and form handling.
- **MySQL**: For database management.
